# Flight Trajectory Planning and Performance Analysis (ATL → ADD)

## Project Description

This project focuses on planning and analyzing a complete flight trajectory between two specified airports: Hartsfield–Jackson Atlanta International Airport (ATL) and Addis Ababa Bole International Airport (ADD).

The flight model includes all major flight phases, from take-off to landing, enabling a comprehensive evaluation of aircraft performance throughout the mission.

## Scope of the Project

The simulation covers the following flight phases:

* Take-off
* Climb
* Cruise
* Descent
* Landing

Each phase is modeled and analyzed to evaluate performance characteristics such as efficiency, altitude profile, and overall flight behavior.
